# Pip Card–Decan Reference Table

## How to Read This Table
Each pip card has TWO planetary rulers:
- **Sephirotic ruler** (pip number) = the mode of force, HOW it operates
- **Decan ruler** (Chaldean cycle) = the operating planet, WHO is acting

When both rulers are the same planet: **double emphasis** (intensified, marked with ⚡)
The suit = element = WHERE it operates (Wands/Fire, Cups/Water, Swords/Air, Pentacles/Earth)

### Modality Key (Earth axis discovery)
- **Cardinal** (pips 2,3,4) = **2** = growth, initiation
- **Fixed** (pips 5,6,7) = **5** = station, pivot, holding center
- **Mutable** (pips 8,9,10) = **8** = seed of transformation, dissolution

This is the second half of the zodiac's waning arc — where the cycle intensifies toward completion.

## Chaldean Cycle Quick Reference
Starting from Aries 1st decan, the endless cycle:
**♂ → ☉ → ♀ → ☿ → ☽ → ♄ → ♃ → (repeat)**

Each sign gets 3 consecutive rulers. The starting ruler repeats every 7 signs:
- Aries/Scorpio: ♂ Mars
- Taurus/Sagittarius: ☿ Mercury
- Gemini/Capricorn: ♃ Jupiter
- Cancer/Aquarius: ♀ Venus
- Leo/Pisces: ♄ Saturn
- Virgo: ☉ Sun
- Libra: ☽ Moon
